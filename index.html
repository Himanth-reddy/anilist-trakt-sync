This project automatically syncs your scrobble history from Anilist to Trakt.

This page is for setup and manual runs.
The automatic sync runs once per day.

1. Get Trakt Access Token (One-Time Setup) 2. Run Manual Sync
One-Time Setup Instructions:

1. Go to your new Vercel project's **Settings -> Environment Variables**.

2. Add all your keys from your local `.env` file (like `TRAKT_CLIENT_ID`, `TRAKT_CLIENT_SECRET`, `SIMKL_CLIENT_ID`).

3. Add your **`ANILIST_ACCESS_TOKEN`** (the long one from your local tests).

4. In the HTML for this file, find the link for "Get Trakt Access Token" and **replace** `YOUR-NEW-PROJECT-URL` with your actual Vercel URL.

5. Click the "Get Trakt Access Token" button. It will give you a JSON page. Copy the `access_token` value.

6. Go back to Vercel and add a new variable: **`TRAKT_ACCESS_TOKEN`**. Paste the token there.

7. In Vercel, go to the "Deployments" tab and **Redeploy** the project to load all the new tokens.
